# PortSwigger Labs Documentation

## Description

This repository contains a detailed record of completed PortSwigger Web Security Labs. It tracks progress across Apprentice, Practitioner, and Expert difficulty levels while providing a structured overview of vulnerabilities, tools used, and mitigation strategies. This documentation serves as a personal reference and learning portfolio for deepening understanding of web application security.

## Objective

To deepen my understanding of web security by carefully completing PortSwigger labs and recognizing recurring vulnerability patterns.
